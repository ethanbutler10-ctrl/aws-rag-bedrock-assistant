# Amazon DynamoDB

## What It Is
Amazon DynamoDB is a fully managed NoSQL database service that provides fast and predictable performance with seamless scalability.

## Key Features
- Fully managed database
- Low latency performance
- Automatic scaling
- Built-in security and backups
- Serverless architecture

## Common Use Cases
- Web applications
- Gaming leaderboards
- IoT data storage
- Real-time analytics

## Difference Between DynamoDB and RDS
DynamoDB is a NoSQL database optimized for key-value data.  
RDS is a relational database designed for structured data using SQL.
