# Amazon EC2

## What It Is
Amazon EC2 (Elastic Compute Cloud) is a cloud computing service that provides virtual servers called instances. These instances allow users to run applications in the cloud without managing physical hardware.

## Key Features
- Scalable virtual machines
- Multiple instance types
- Custom operating systems
- Security groups for firewall rules
- Integration with many AWS services

## Common Use Cases
- Web hosting
- Application servers
- Machine learning workloads
- Development environments
- High-performance computing

## Difference Between EC2 and Lambda
EC2 requires managing servers and operating systems.  
Lambda is serverless and runs code automatically without managing infrastructure.
