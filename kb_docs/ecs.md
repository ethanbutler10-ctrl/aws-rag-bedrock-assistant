# Amazon ECS

## What It Is
Amazon ECS (Elastic Container Service) is a managed container orchestration service that allows users to run and manage Docker containers on AWS.

## Key Features
- Managed container orchestration
- Integration with AWS services
- Supports EC2 and Fargate launch types
- Scalable container deployments

## Common Use Cases
- Running microservices
- Deploying containerized applications
- Managing backend services

## Difference Between ECS and EKS
ECS is AWS’s native container orchestration service.  
EKS is a managed Kubernetes service.
