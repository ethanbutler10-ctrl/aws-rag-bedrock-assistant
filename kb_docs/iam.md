# AWS IAM

## What It Is
AWS Identity and Access Management (IAM) is a service that controls access to AWS resources. It allows administrators to create users, groups, and roles with specific permissions.

## Key Features
- Role-based access control
- Policy-based permissions
- Multi-factor authentication
- Secure access management
- Integration with AWS services

## Common Use Cases
- Managing user access
- Granting service permissions
- Securing AWS environments
