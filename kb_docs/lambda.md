# AWS Lambda

## What It Is
AWS Lambda is a serverless compute service that allows developers to run code without managing servers. Lambda automatically scales and only runs when triggered by events.

## Key Features
- Serverless architecture
- Automatic scaling
- Event-driven execution
- Pay only for compute time used
- Supports multiple programming languages

## Common Use Cases
- Backend APIs
- Data processing
- File processing
- Automation tasks
- Microservices

## Difference Between Lambda and EC2
Lambda runs code in response to events and does not require server management.  
EC2 requires users to manage virtual servers.
