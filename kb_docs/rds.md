# Amazon RDS

## What It Is
Amazon RDS (Relational Database Service) is a managed database service that supports relational databases such as MySQL, PostgreSQL, Oracle, and SQL Server.

## Key Features
- Automated backups
- Managed database updates
- High availability with Multi-AZ deployments
- Read replicas for scaling
- Built-in monitoring

## Common Use Cases
- Web applications
- Enterprise databases
- E-commerce systems
- Customer data storage

## Difference Between RDS and DynamoDB
RDS is a relational database using SQL and structured tables.  
DynamoDB is a NoSQL database optimized for key-value workloads.
