# Amazon VPC

## What It Is
Amazon Virtual Private Cloud (VPC) allows users to create a private network inside AWS. It enables control over IP ranges, subnets, routing, and security.

## Key Features
- Private network isolation
- Subnets and routing tables
- Security groups and network ACLs
- Internet gateways and NAT gateways

## Common Use Cases
- Hosting secure applications
- Network segmentation
- Hybrid cloud connectivity
